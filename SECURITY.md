# Güvenlik politikası

## Desteklenen sürümler

| Sürüm | Destek |
| --- | --- |
| 0.3.x | Evet |
| 0.2.x ve öncesi | Hayır |

## Güvenlik açığı bildirme

Bir güvenlik açığı bulursanız herkese açık issue açmayın. GitHub deposundaki
**Security → Report a vulnerability** bölümünü kullanarak özel bildirim gönderin.

Bildirimde mümkünse etkilenen sürümü, tekrar oluşturma adımlarını, olası etkileri
ve önerilen düzeltmeyi belirtin.

API anahtarlarını, erişim tokenlarını veya gerçek proje sırlarını bildirime eklemeyin.
